Projeto para realizar as operações CRUD

1)  Criar um banco de dados com a tabela USUARIO com os seguintes campos

login, senha, nome e email

2) Criar um documento JSP com cadastro de usuário com os seguintes campos:

Login  (duas pessoas não pode ter a mesmo login)

Nome completo

Email

Confirmar de Email

Senha      (mínimo de 4 e máximo de 8 carateres)

Confirmar senha

3) Criar um Servlet  para realizar as seguintes validações

a) Não pode ter Login repetido na tabela 

b) Senha diferente do login  (mínimo de 4 e máximo de 8 carateres)

c) Usuário deve confirmar o email e senha

4) Criar um documento JSP/Servlet para lista todos os usuários cadastrados

5) Criar um documento JSP/Servlet para realização de login

6) Criar um documento JSP/Servlet para atualizar dos dados de um usuário

7) Criar um documento JSP/Servlet para remover usuário cadastrado
